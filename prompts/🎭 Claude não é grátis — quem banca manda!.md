# Prompt para Nano Banana Pro
Aqui vai o **prompt direto**, simples, humano, sem texto na imagem:

```
Crie uma imagem em realismo dramático.

Primeiro plano:
Meu rosto em close, expressão séria e desconfortável, olhando para frente.
IMPORTANTE: respeitar meu topete exatamente como é, não alterar cabelo.

Segundo plano:
A palavra “CLAUDE” representada como um boneco de marionete, sólida, pesada.
Cordas de marionete vindo de cima, como se alguém estivesse controlando a palavra.
A mão que controla pode aparecer parcialmente ou só sugerida pelas cordas.

Iluminação:
Cores quentes, tons de laranja, vermelho e dourado.
Luz dramática, alto contraste, clima de tensão e poder.
Reflexo da luz quente no meu rosto.

Estilo:
Realismo fotográfico.
Nada cartunesco.
Nada infantil.
Nada futurista exagerado.

Composição:
Plano fechado para funcionar bem em mobile.
Espaço negativo limpo para possível GC depois.
Foco no rosto + na palavra CLAUDE como objeto controlado.

Não gerar nenhum texto adicional na imagem.
```

Se quiser depois, ajustamos **expressão facial** (mais raiva / mais alerta) ou **quantidade de cordas** para calibrar o impacto no feed.
